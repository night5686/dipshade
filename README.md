## 🚀 Deployment on Vercel

### Quick Deploy
1. Upload all files to your repository
2. Connect to Vercel
3. Deploy instantly!

### Manual Deploy
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the website directory
3. Follow the prompts

```css
    --accent-gold: #ffd700;
    /* ... more variables */
}
```
